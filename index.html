Sobre as Tags

As Tags são usadas para demarcar começo ou fim de um elemento HTML dentro de um documento do mesmo tipo. Segundo o site Digital Ocean,

"As tags também ajudam navegadores na converção de documentos HTML para suas respectivas páginas web"
. Neste contexto, podemos mesmo citar a comumente usada tag "p" cujo uso é de criar parágrafos para organização textual, ou ainda, "strong" cuja função é tornar a palavra referenciada em negrito.

E quanto aos Atributos das Tags?

Enquanto as Tags cuidam da parte de demarção dos elementos HTML, os atributos funcionam como formas de marcação para alterar como as Tags se comportam. Ainda segundo o site w3school

"Atributos fornecem informações adicionais sobre elementos, devem sempre ser especificados no começo da tag e são comumente seguidos em pares de name/value onde name="value"."
Por exemplo, atributos como o "color" podem ser usados para alterar a cor de fonte, enquanto "weight" é um atributo usado para alterar a largura de um elemento.

Textos em HTML

Funcionando de forma similar aos similar aos atributos, documentos HTML também possuem formatos de formatação de texto que vão desde coisas simples como demarcação de parágrafo até menções com "blockquote", cerrilhado com "del" ou mesmo subscritos e superscripto com "sub" e "sup" respectivamente.

Os Links

Links como o próprio nome sugere, são lugares para onde direcionares o usuário após clicar em determinado lugar, para tal, usamos a letra "a" para definir o hyperlink ao qual será usado seguido de href dando assim, a referência que será usada.
